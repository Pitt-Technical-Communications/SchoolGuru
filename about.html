About




The SchoolGuru website helps student figure out when to study for the many assignments that they have planned over the next week.

Use the add assignment button to input assignments, tests, readings, ect.

Then with the given information about the tasks, the site will prioritize when you should work on each task throughout the week.

You can then export the week view of work to an external source for later reference.

Note: You must put in all known assignments, as you cannot recall data from a previous session on this site.
